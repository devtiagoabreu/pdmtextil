### Compreensão do Projeto:

O **PDM Pro Textil (Pro Moda)** é um sistema web abrangente desenhado para centralizar e digitalizar a gestão de desenvolvimento de produtos têxteis, operando como o elo de comunicação entre os departamentos Comercial, Tecelagem, Beneficiamento e PCP. O sistema substitui fluxos descentralizados baseados em e-mail e planilhas por uma plataforma que organiza o briefing de desenvolvimento, acompanha status de cada solicitação, estrutura dados de fichas técnicas para tecidos cru e beneficiados, roteiros de produção, gerando ao final os códigos dos produtos rigorosamente em conformidade com o ERP Systêxtil.

**Tecnologias Envolvidas:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS e shadcn/ui
- Drizzle ORM + Neon PostgreSQL (suporte dinâmico para banco de dados local Postgres)
- NextAuth.js (sistema de perfis e permissões)
- react-hook-form + Zod para manipulação e validação de dados
- TanStack Query & Table
- Vercel (Hospedagem) e Vercel Blob (Upload de Anexos)
